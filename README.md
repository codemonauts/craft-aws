## Requirements

 * Craft CMS >= 3.0.0

## Installation

Open your terminal and go to your Craft project:

``` shell
cd /path/to/project
composer require codemonauts/craft-aws
```

In the control panel, go to Settings → Plugins and click the “install” button for *Craft AWS*.
